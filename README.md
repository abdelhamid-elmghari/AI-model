#  AI Emotion Detector

A modern web application that detects emotions from live webcam input or uploaded images using a **Teachable Machine Image Model**. The project is built with **HTML, CSS, and JavaScript**, leveraging **TensorFlow.js** and **Teachable Machine** for real-time predictions.

---

##  Features

- Real-time emotion detection from webcam.
- Upload images for emotion prediction.
- Snapshot feature to save frames.
- Animated prediction bars showing confidence percentages.
- Pause/Resume webcam stream.
- Responsive and modern UI with a popup introduction.
